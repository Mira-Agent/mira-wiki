# Slack

Connect your Slack workspace and communicate through Mira.

## Connect

```
Connect my Slack
```

## Use cases

### Send messages

```
Send a message to #general: "Team standup in 10 minutes"
Post in #design: "New mockups are ready for review"
```

### Search & read

```
What's new in #engineering?
Search Slack for "deploy" in #devops
Summarize today's messages in #general
```

### Channels

```
List channels I'm a member of
What channels exist in the workspace?
```

<Tip>Mira can bridge Telegram and Slack — read updates in one and post in the other.</Tip>
