# Tools & Integrations

Mira connects to the apps you already use. Send emails, manage calendars, create issues, update spreadsheets — all from your Telegram chat.

## How to connect

Just tell Mira what you want to connect:

```
Connect my Gmail
Set up Google Calendar
Link Notion
```

Mira will send you a secure OAuth link. Click it, authorize, and you're done. No API keys, no config files, no terminal.

## Popular use cases

### Email management

```
Check my unread emails
Send an email to John about the meeting tomorrow
Summarize the last 5 emails from my boss
```

### Calendar & scheduling

```
What's on my calendar today?
Create a meeting with Sarah for Friday at 2pm
Find a free slot next week for a 1-hour call
```

### Project management

```
Create a Linear task: "Fix login bug on mobile" with priority High
What are my open GitHub issues?
Update the Notion database with today's metrics
```

### Content & marketing

```
Post on LinkedIn: "Excited to announce our new feature!"
Create a Canva design for an Instagram story
Add a row to the Google Sheet with campaign results
```

### Development

```
Create a GitHub issue in mira-wiki repo
Deploy the latest commit to Vercel
List open PRs in the main branch
```

## Security

<Info>
  All connections use OAuth 2.0 — the same standard used by Google and Microsoft. Mira never sees or stores your passwords. You can revoke access at any time from the app settings.
</Info>

## Disconnecting

```
Disconnect Gmail
Remove my Notion connection
```

Mira will remove the connection immediately.
