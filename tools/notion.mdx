# Notion

Connect your Notion workspace and manage it through Mira.

## Connect

```
Connect my Notion
```

## Use cases

### Create pages

```
Create a Notion page called "Meeting Notes" in my Work database
Add a new entry to my Tasks database: "Fix login bug" with priority High
```

### Search & read

```
Search my Notion for "Q4 roadmap"
What's in my Tasks database?
Show me pages created this week
```

### Update entries

```
Mark the task "Design review" as done
Update the status of "Launch v2" to In Progress
```

<Tip>Notion works best when you tell Mira the exact database name — e.g. *"Add to my 'Sprint Board' database"*.</Tip>
