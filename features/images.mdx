# Image Generation & Editing

Describe what you want and Mira will generate it. Or send a photo and ask her to transform it.

## Generate images

Costs **30 tokens** per image.

```
Generate a portrait of a woman in Art Nouveau style, floral background
```

### Models

Mira offers multiple image generation models, each optimized for different use cases:

| Model | Best for | Cost |
|-------|----------|------|
| **GPT Image 2** | Images with text, high detail, cards, posters, slides with readable text | 30 🪙 |
| **Nano Banana** | Quick sketches, illustrations, balanced quality & creativity | 30 🪙 |
| **Nano Banana Pro** | Hero visuals, marketing materials, highest quality | 60 🪙 |
| **Seedream 4.5** | Portrait photos, face similarity, realistic faces | 30 🪙 |
| **Qwen Image Edit** | Low moderation, creative freedom, artistic experiments | 30 🪙 |

### Aspect ratios

| Ratio | Best for |
|-------|----------|
| `9:16` | Stories, vertical content (default) |
| `16:9` | Landscapes, presentations |
| `1:1` | Profile pictures, posts |
| `4:5` | Instagram feed |
| `3:2` | Photography style |

### Batch generation

Ask for multiple images at once:

```
Generate 3 variations of a logo for a coffee shop
Create 4 slides for a pitch deck about AI
```

## Edit images

Send any photo to Mira and ask her to transform it. Costs **30 tokens**.

**Examples:**
- *"Make this photo anime style"*
- *"Add a forest background"*
- *"Change the color palette to warm tones"*
- *"Make it look like a painting"*

<Tip>You can send multiple images and ask Mira to combine or compare them.</Tip>

## Analyze images

Send a photo and ask Mira what's in it — **free of charge**.

```
What's in this photo?
Read the text on this sign.
Describe the mood of this scene.
```
