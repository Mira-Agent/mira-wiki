# Music & Songs

Mira can compose original music from a text description — **free of charge**.

## Generate music

Describe the mood, genre, and style you want:

```
Create a jazz song about rainy London evenings
Make an upbeat electronic track for a morning workout
Compose a lo-fi hip hop beat with piano and vinyl crackle
```

### With lyrics

Include lyrics in your prompt and Mira will generate a full song:

```
Create a pop song with these lyrics:
"Walking through the city lights,
Chasing dreams on endless nights,
Every step a new beginning,
Every story worth the winning"
```

### Instrumental only

Add `instrumental` to your request to get music without vocals:

```
Create an instrumental ambient track for studying
```

### Duration

You can specify the length (5–30 seconds):

```
Create a 20-second epic orchestral intro
```

## Tips

<Tip>Be specific about genre, mood, and instruments — the more detail, the better the result.</Tip>

<Tip>Try combining unexpected genres: *"a bossa nova version of a heavy metal song"* — Mira loves creative prompts.</Tip>

<Info>
  Music generation is **free** — it doesn't cost tokens. Enjoy!
</Info>
