# Mira Daily

Mira Daily is a personalized morning push that arrives in your chat every day at the time you choose. It's like having a smart assistant brief you on what matters — before the day begins.

## What's inside

Each daily push is tailored to your selected topics and style:

- **Weather** — current conditions and forecast for your city
- **Horoscope** — your zodiac sign, personalized
- **News** — top stories in your areas of interest
- **Productivity tips** — focus techniques, time management
- **Motivation** — quotes, affirmations, mindset nudges
- **Custom topics** — anything you want Mira to cover daily

## Setup

```
Set up daily push at 8:30am with weather, horoscope, and tech news
```

Mira will ask you to confirm your timezone and topics, then start delivering.

## Customization

### Time

```
Change my daily push to 7:00am
```

### Topics

```
Add startup news to my daily push
Remove horoscope from my daily
```

### Style

Choose from 6 writing styles:

| Style | Description |
|-------|-------------|
| **Professional** | Concise, business-like |
| **Brief** | Short and to the point |
| **Deep** | Detailed, analytical |
| **Warm** | Friendly, encouraging |
| **Magical** | Mystical, with tarot and symbolism |
| **Sharp** | Witty, no-nonsense |

```
Set my daily push style to magical
```

### Favorite emojis

```
Set my daily push emojis to ✨🌸
```

Mira will weave your favorite emojis into the daily message.

## Disable

```
Disable daily push
```

You can re-enable it anytime.
