# API Reference

<Badge variant="secondary">Coming Soon</Badge>

The Mira API will allow developers to integrate Mira's AI capabilities directly into their own applications, bots, and services.

## What's coming

- **Chat API** — send messages and receive AI responses programmatically
- **Memory API** — read and write user memory and context
- **Generation API** — generate images, videos, and music via HTTP
- **Skills API** — create, manage, and execute custom skills
- **Integrations API** — trigger connected app actions (Gmail, Calendar, Notion, etc.)
- **Webhooks** — receive real-time events when triggers fire

## Use cases

- Build custom Telegram bots powered by Mira
- Embed Mira's AI into your product
- Automate workflows across multiple services
- Create agent-to-agent communication protocols
