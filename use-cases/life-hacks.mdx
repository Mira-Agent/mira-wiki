# Life Hacks

These are the hidden gems — clever, unexpected ways to use Mira that save time and make life easier.

## Voice as your superpower

Mira's voice interface is the fastest way to interact with AI. No typing, no waiting:

- **Morning commute** — dictate tasks, check emails, plan the day while driving
- **Cooking** — ask for recipe substitutions without touching your phone
- **Walking** — brainstorm ideas out loud, Mira transcribes and organizes
- **Before sleep** — dictate tomorrow's to-do list with your eyes closed

## Memory tricks

```
Remember that I prefer bullet points over paragraphs
Remember that I'm allergic to shellfish
Remember that my meeting with Sarah is every Tuesday at 3pm
```

Mira stores these facts and applies them in every future conversation.

## Group chat superpowers

- **Summarize 200+ messages** — *"What happened while I was away?"*
- **Track decisions** — *"What did we decide about the launch date?"*
- **Assign tasks** — *"Remind @alex about the design review by Friday"*
- **Onboard new members** — *"Summarize the last month in this chat"*

## Image tricks

- **Try on clothes** — send a selfie + a clothing photo, ask Mira to try it on
- **Color type analysis** — send a selfie, get personalized color recommendations
- **Batch generation** — *"Generate 4 variations of a logo"*
- **Photo restoration** — *"Make this old photo look new"*

## Automation hacks

### Recurring reminders

```
Remind me every Monday at 9am to plan the week
Remind me on the 1st of every month to review finances
Remind me every Friday at 5pm to wrap up
```

### Daily digest

```
Set up daily push at 8:30am with weather, horoscope, and tech news
```

### Skills as shortcuts

```
Create a skill called "Standup" that:
- Lists my Linear tasks due this week
- Summarizes my calendar for today
- Generates a standup message
```

Now just type `/standup` every morning.

## Integration combos

### Gmail + Calendar

```
Check my emails and add any meeting requests to my calendar
```

### Linear + Slack

```
Create a Linear task from this Slack message and notify #engineering
```

### GitHub + Notion

```
When a PR is merged, update the Notion database
```

## Quick wins

- **Translate** — *"Translate this to French"* — any language, instant
- **Summarize** — send a PDF, get a 5-bullet summary
- **Calculate** — *"If we grow 15% MoM, what's our revenue in 12 months?"*
- **Compare** — *"Compare these two job offers"*
- **Brainstorm** — *"Give me 20 startup ideas in the pet industry"*
- **Draft** — *"Draft a polite decline for this event invitation"*
