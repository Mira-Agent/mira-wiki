## Time-saving workflows

### Concept generation

```
Generate a landing page hero image for a meditation app
Create 3 variations of a logo for a coffee brand
Design a poster for a music festival — neon, retro style
```

### Design iteration

```
Make this image more minimal
Change the background to a gradient from blue to purple
Convert this photo to a flat illustration style
```

### Research & inspiration

```
What are the current UI design trends for 2026?
Find examples of brutalist web design
Suggest typography pairings for a luxury brand
```

### Documentation

```
Create a design brief from this client message
Write a handoff document for this component
Generate Figma annotations for this layout
```

## Popular use cases

- **Style transfer** — transform photos into illustrations, paintings, or specific art styles
- **Icon generation** — create consistent icon sets for any project
- **UX review** — analyze screenshots for accessibility and usability
- **Brand kit creation** — generate color palettes, typography, and visual language
- **Client communication** — draft design briefs, feedback responses, and proposals
- **Presentation design** — generate slide visuals and layouts
- **Responsive mockups** — generate variations for mobile, tablet, desktop
- **Design system docs** — document components, tokens, and patterns
